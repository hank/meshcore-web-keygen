MeshCore Key Generator

Generate custom Ed25519 keys for MeshCore

About MeshCore Keys

  • Meshcore uses the first two characters of your public key as a node identifier. It is helpful to be able to select your node id to avoid collisions with neighbors.
  • This key generator does all processing in your browser - keys never leave your device

📋 How to Import Keys into MeshCore

×

🔧 For Companion Nodes

  1. Connect to your node using the MeshCore app
  2. Tap the Settings gear icon
  3. Tap Manage Identity Key
  4. Paste your Private Key into the text box
  5. Tap Import Private Key
  6. Important: Tap the checkmark ✓ to save changes in settings

💻 For Repeater + Computer

If you can connect your repeater directly to a computer:

  1. Connect to your repeater via USB serial
  2. Open the MeshCore Web Console or use any terminal
  3. Run the command: set prv.key <your_private_key>
  4. This will change the device's private key directly

📡 For Repeater + Companion

You can set the private key on a repeater remotely over LoRa:

  1. Select Repeater from Contact List
  2. Enter Password and Log into Repeater
  3. Click on Command Line option (at the bottom of the repeater info screen)
  4. Enter set prv.key <hex> and press Enter
  5. Reboot the repeater after set prv.key <hex> for the new private key to take effect
